Scenario 1: E-commerce Sales Data Analysis

Dataset: https://www.kaggle.com/datasets/carrie1/ecommerce-data

This scenario involves analyzing an e-commerce transactions dataset to explore product categories, sales behavior, and data quality. The dataset is loaded into Pandas and examined using basic exploratory functions such as head(), info(), and describe(). Missing values are checked to evaluate data completeness. Bar charts and line plots are used to visualize sales per product and identify overall sales trends.

Scenario 2: Hospital Patient Records Analysis

Dataset: https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database

In this scenario, patient health records are analyzed to detect missing medical attributes and understand key health patterns. The dataset structure and null values are explored after loading it into Pandas. Histograms and boxplots are used to visualize glucose levels and age distribution, helping identify variability, outliers, and possible indicators related to diabetes risk.

Scenario 3: Housing Dataset Analysis

Dataset: https://www.kaggle.com/datasets/harishkumardatalab/housing-price-prediction

This scenario focuses on examining a housing dataset to identify missing features and analyze relationships between housing attributes and prices. After inspecting the dataset and summarizing missing values, scatter plots and correlation heatmaps are used to visualize relationships among numerical variables. The analysis highlights important factors influencing house prices and supports price prediction insights.

Scenario 4: Customer Personality Analysis

Dataset: https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis

This scenario analyzes customer demographic and behavioral data to identify missing account information and understand customer characteristics. The dataset is explored for structure, data types, and null values. Bar charts and boxplots are used to visualize age, income, and spending patterns. The results provide insights into customer segments and support data-driven marketing strategies.
